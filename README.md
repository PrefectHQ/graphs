# Prefect Graphs
Large scale graphs designed for Prefect

## Demo
https://prefect-graphs.netlify.app/

## Development
```
nvm use
npm ci
npm run serve
```

## Installation
```
npm i @prefecthq/prefect-graphs --save
```
